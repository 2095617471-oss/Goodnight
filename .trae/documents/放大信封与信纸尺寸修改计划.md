# 放大信封与信纸尺寸修改计划

## 1. 目标与需求分析
用户希望将“时光机信封”及其内部的“信纸”放大到当前尺寸的两倍，同时进一步调整信封样式，使其更加逼真。

## 2. 现状分析
- **涉及文件**：`css/style.css`
- **当前尺寸**：
  - 信封容器 `.letter-popup-container`：宽 `400px`，高 `260px`。
  - 信纸 `.envelope-paper`：宽 `340px`，高 `480px`。
- **目标尺寸（两倍）**：
  - 信封容器：宽 `800px`，高 `520px`。
  - 信纸：宽 `680px`，高 `960px`。

## 3. 具体修改方案

### 3.1 尺寸调整 (`css/style.css`)
- **信封容器** `.letter-popup-container`：
  - `width: 800px;`
  - `height: 520px;`
- **信纸** `.envelope-paper`：
  - `width: 680px;`
  - `height: 960px;`
  - 相应增加内边距以适应大尺寸：`padding: 80px 60px;`

### 3.2 字体与排版适配调整 (`css/style.css`)
随着容器的放大，内部的元素也需要成比例放大，否则会显得空旷：
- **信封正面 (`.envelope-front`)**：
  - 图标 `i`：`font-size: 100px;`，`margin-bottom: 30px;`
  - 标题 `h3`：`font-size: 40px;`，`margin: 0 0 10px 0;`
  - 副标题 `p`：`font-size: 28px;`
- **信纸内部 (`.envelope-paper`)**：
  - 落款日期 `.letter-header`：`font-size: 28px;`，`margin-bottom: 50px;`，`padding-bottom: 20px;`
  - 正文 `.letter-content`：`font-size: 32px;`，`line-height: 2;`，`margin-bottom: 50px;`
  - 按钮 `.paper-btn`：`font-size: 28px;`，`padding: 20px 40px;`

### 3.3 动画参数适配调整 (`css/style.css`)
由于尺寸变大，部分位移动画参数也需要等比放大，以确保动画效果一致：
- `.shake-in` 动画中的 `floatIn`：`translateY(80px)` 调整为 `translateY(160px)`。
- `.letter-envelope.opened`：
  - `transform: translateY(120px) scale(0.9);` 调整为 `transform: translateY(240px) scale(0.9);`
- `.letter-envelope.opened .envelope-front`：
  - `transform: translateY(20px);` 调整为 `transform: translateY(40px);`
- `.letter-envelope.opened .envelope-paper`：
  - `transform: translate(-50%, calc(-50% - 100px)) scale(1);` 调整为 `transform: translate(-50%, calc(-50% - 200px)) scale(1);`

### 3.4 增加真实信封细节 (`css/style.css`)
为 `.letter-envelope` 增加一个信封折痕的效果（利用 `::before` 和 `::after` 伪元素）：
- 添加一个顶部的三角形/多边形阴影，模拟信封翻盖。
- 添加底部的斜线阴影，模拟信封折叠的接缝。

## 4. 验证与测试
- 触发一封未来的信件弹出。
- 检查信封尺寸是否显著变大（800x520），文字图标是否适配，并带有真实的折痕/翻盖视觉效果。
- 检查悬浮动画是否自然。
- 点击拆开信封，观察信纸滑出动画是否居中，且不被过度裁切。
- 检查信纸内部排版是否等比例放大，清晰易读。
