# 优化「时光机信封」与「信件展示」样式修改计划

## 1. 目标与需求分析
根据您的要求，本次修改将对“时光机信封”弹窗的视觉效果、信件纸张的样式以及交互动效进行全面升级。整体风格将向真实物理质感靠拢，采用温暖的米棕色调，并加入柔和的过渡动画，营造出“拆信”的仪式感。同时，保持原有的弹窗逻辑和事件绑定不变。

## 2. 现状分析
- 涉及文件：`index.html`, `css/style.css`
- 当前情况：
  - 信封图标使用的是线性图标 `ri-mail-send-line`。
  - 信封容器 (`.letter-popup-container`) 尺寸偏小 (320x220)。
  - 信纸 (`.envelope-paper`) 呈现现代卡片/便签样式，没有真实信纸质感。
  - 打开动画是简单的位移，稍显生硬。

## 3. 具体修改方案

### 3.1 修改 HTML 结构 (`index.html`)
- 将信封图标由 `<i class="ri-mail-send-line envelope-icon"></i>` 替换为更加写实的 `<i class="ri-mail-fill envelope-icon"></i>`。
- 其他结构保持不变，以确保原有的 JS 逻辑正常运行。

### 3.2 优化信封外观 (`css/style.css`)
- **尺寸与质感**：
  - 将 `.letter-popup-container` 的尺寸扩大至 `400px * 260px`，使其比例更接近真实信封。
  - `.letter-envelope` 背景修改为米棕色/牛皮纸色的渐变 (`linear-gradient(145deg, #d5c3aa, #bca382)`)。
  - 增加边缘内阴影和更深的外部投影，模拟真实的厚度和光影质感。
- **排版与颜色**：
  - `.envelope-front` 内的图标、标题、副标题修改为深棕色 (`#5c4a3d`)，字体调柔和。
  - 为 `.envelope-front` 增加一个轻微上下浮动的动画，营造“悬浮/待拆封”的仪式感。

### 3.3 优化信件内容页（A4信纸风格） (`css/style.css`)
- **信纸尺寸与背景**：
  - 调整 `.envelope-paper` 的尺寸为 `340px * 480px`（类似A4比例）。
  - 背景色改为米白/奶白色 (`#fdfbf7`)，并加入轻微做旧的内阴影 (`inset 0 0 40px rgba(139,115,85,0.15)`)。
- **排版优化**：
  - `.letter-header` (写于 xxxx-xx-xx) 调整为信纸顶部的落款风格，右对齐或居中对齐，深灰色/浅棕色。
  - `.letter-content` 字体变柔和，行间距加大 (`line-height: 2`)，文字颜色使用深棕黑 (`#4a3f35`)。
  - 修改 `#close-letter-btn` 按钮样式，使其像一个信封邮戳或牛皮纸按钮（带边框、米白色背景、棕色文字），与信纸风格高度统一。

### 3.4 优化过渡动效 (`css/style.css`)
- 将原本稍显抖动的 `gentleShake` 动画替换为更平滑舒缓的浮动动画 `envelopeFloat`。
- 优化 `.letter-envelope.opened` 状态下的过渡：
  - 拆信时，信封本体向下退出并淡出。
  - 信纸从信封中心优雅地滑出、放大，并停留在屏幕中央。

## 4. 验证与测试
- 触发一封未来的信件弹出（或在控制台手动调用 `showLetterPopup(msg)`）。
- 检查未拆开时的信封外观：尺寸、质感、图标、悬浮动画、颜色是否符合预期。
- 点击拆开信封，观察过渡动画是否自然。
- 检查拆开后的信纸样式：是否符合A4米白信纸、做旧边缘、文字排版是否舒展。
- 点击“收下这份心意”按钮，确认可以正常关闭弹窗。
