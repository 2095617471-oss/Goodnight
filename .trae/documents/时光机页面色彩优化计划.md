# 时光机页面UI色彩优化计划

## 1. 目标与需求分析
用户反馈“时光机那一页太丑了，优化一下增加点色彩”。
目前时光机页面主要由标题“时光机”和一个包含寄信表单（`.future-card` / `.envelope-form`）及留言列表（`.future-messages-list`）的卡片组成。整体配色偏暗黑（`--bg-color`, `--surface-color`），通过边框和极少量的 `rgba(255,255,255,0.02)` 区分层级，视觉上显得过于单调和冷清。

由于“时光机”带有“寄信给未来”、“温暖期待”的情感属性，我们将引入与之前“信封弹窗”相呼应的温暖色调（米棕色/牛皮纸色），打破现有的黑灰单调，增加页面的治愈感和复古色彩。

## 2. 具体修改方案

### 2.1 整体卡片与表单色彩优化 (`css/style.css`)
- **表单容器 `.envelope-form`**：
  - 将背景颜色从暗淡的半透明白改为带有温暖气息的暗棕色渐变：`background: linear-gradient(135deg, rgba(139,115,85,0.1), rgba(100,70,40,0.05));`。
  - 边框颜色改为偏棕色的：`border: 1px dashed rgba(139,115,85,0.4);`。
  - Hover 状态：边框变亮 `rgba(139,115,85,0.8)`，背景微亮 `rgba(139,115,85,0.15)`。
- **邮票装饰 `.envelope-stamp`**：
  - 边框改为：`border: 2px dashed rgba(139,115,85,0.5);`
  - 邮票图标颜色改为：`color: rgba(139,115,85,0.6);`
- **输入框与文本域 (`.future-form input, .future-form textarea`)**：
  - 背景色略微区分于纯黑背景，带点暖色调：`background-color: rgba(255,251,245,0.03);`
  - 边框：`border: 1px solid rgba(139,115,85,0.2);`
  - 聚焦状态 (focus)：边框高亮为 `rgba(139,115,85,0.6)`。

### 2.2 留言列表卡片色彩优化 (`css/style.css`)
- **普通已送达卡片 `.future-message-card`**：
  - 左侧边框颜色改为温暖的橙棕色：`border-left: 3px solid #d5c3aa;`
  - 背景色增加极轻微的暖调：`background-color: rgba(213,195,170,0.05);`
- **封存状态卡片 `.future-message-card.sealed`**：
  - 左侧边框改为深灰棕色：`border-left-color: #6d5b4b;`
  - 斜纹背景（胶带封条效果）改为更具色彩感的棕色斜纹：
    ```css
    background: repeating-linear-gradient(
        -45deg,
        rgba(109,91,75,0.05),
        rgba(109,91,75,0.05) 10px,
        rgba(109,91,75,0.1) 10px,
        rgba(109,91,75,0.1) 20px
    );
    ```

### 2.3 按钮与标题修饰
- 时光机模块的 `<h3>` 标题加上一点色彩，图标颜色使用主色（Accent color）或温暖的米色，使其更加醒目。

## 3. 验证与测试
- 切换到“时光机”Tab。
- 检查“给未来的自己”表单区域，背景是否呈现温暖的棕色调，邮票装饰是否清晰。
- 点击输入框，验证边框聚焦颜色。
- 投递一封信件（未来日期），检查下方出现的“已封存”状态卡片的棕色斜纹背景效果。
- （如果已有历史数据）检查已送达卡片的暖色调背景和侧边栏颜色。
