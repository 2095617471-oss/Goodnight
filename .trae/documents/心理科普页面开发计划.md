# 「心理科普」页面开发计划

## 一、需求摘要
在现有的“深夜情绪树洞日记”应用中，新增一个独立的「心理科普」导航标签和对应页面。页面主体包含一个垂直排列的文章卡片列表，卡片采用大圆角设计，支持特定的悬停交互效果（放大与阴影加深）。点击卡片后，在当前模块内平滑切换至文章详情视图，保持沉浸式的单页应用体验。

## 二、当前状态分析
- **架构**：纯前端单页应用（SPA），无后端数据库。
- **导航**：通过左侧/底部导航栏切换不同的 `<section class="tab-pane">`。
- **样式**：全局采用暗色护眼模式，已有成熟的变量系统（颜色、圆角、过渡时间等）。
- **数据**：目前日记数据依赖 `localStorage`，而科普文章属于静态配置数据，可以直接内置在 JavaScript 中。

## 三、具体实施方案

### 1. `index.html` 结构修改
- **新增导航项**：在左侧导航栏（`#main-nav`）中添加一项“💡 心理科普”，绑定 `data-tab="tab-edu"`。
- **新增页面容器**：在 `#main-content` 中增加 `<section id="tab-edu" class="tab-pane hidden">`。
- **模块内部分层**：
  - **列表视图 (`#edu-list-view`)**：包含页面顶部标题（“心理科普”），以及用于容纳卡片的容器（`.edu-card-container`）。
  - **详情视图 (`#edu-detail-view`)**：默认隐藏（`hidden`），包含一个“返回”按钮、文章大标题容器以及正文段落容器。

### 2. `css/style.css` 样式修改
- **卡片基础样式 (`.edu-card`)**：
  - 采用大圆角（例如 `border-radius: 16px`）。
  - 使用 Flex 布局，实现左侧主标题、右侧简短正文预览的结构。
  - 背景色采用比全局背景稍亮的卡片底色，文字柔和清晰。
  - 间距适中，确保整体干净舒适。
- **交互动画 (`.edu-card:hover`)**：
  - `transform: scale(1.03)` 实现轻微自然放大。
  - 增加柔和的阴影 `box-shadow`。
  - 设置平滑过渡：`transition: all 0.3s ease`。
- **响应式适配**：
  - 移动端下（`< 768px`），卡片内部的左右布局可自动转为上下堆叠，或者调整字体大小以保证预览文本的可读性。
- **详情页样式**：
  - 简洁的返回按钮样式，文章正文的行高（line-height）、段落间距（margin-bottom）优化，确保长文本阅读体验。

### 3. `js/app.js` 逻辑修改
- **静态数据源**：定义一个 `eduArticles` 数组，存放科普文章的数据（包含 `id`, `title`, `preview`, `content`）。
- **渲染列表**：编写 `renderEduList()` 函数，遍历 `eduArticles` 生成卡片 DOM，并插入到 `#edu-list-view` 中。
- **视图切换逻辑**：
  - 为每个卡片绑定点击事件：调用 `showEduDetail(articleId)`。
  - `showEduDetail(id)`：根据 ID 提取文章内容，渲染到详情视图，隐藏列表视图，显示详情视图。
  - 为详情页的“返回”按钮绑定事件：隐藏详情视图，重新显示列表视图。
- **初始化绑定**：在现有的 `init()` 函数中调用一次 `renderEduList()`。

## 四、假设与决策
- **数据存储**：由于是科普内容，属于产品自带的固定数据，因此将以静态配置形式写在 JS 代码中，方便后续统一维护。
- **切换方式**：根据确认，详情页采用模块内视图切换而非弹窗，以符合当前“极简、不打扰、沉浸”的产品基调。

## 五、验收标准 (Verification)
1. 左侧导航出现“心理科普”选项，点击可正常切换至该页面。
2. 页面内垂直展示多个圆角卡片，左标题右预览，排版清晰。
3. 鼠标悬停卡片时，有 0.3s 的平滑过渡，卡片放大至 1.03 倍并带有柔和阴影。
4. 移动端和电脑端显示均正常（响应式不破版）。
5. 点击任意卡片，平滑进入文章详情页；点击返回按钮，可无缝回到卡片列表。
